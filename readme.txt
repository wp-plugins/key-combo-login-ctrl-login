== Description ==

This WordPress plugin allows you to reach the login page by simply pressing Ctrl + L.

From: aubreypwd / http://aubreypwd.com (formerly known as enethrie).

== Installation ==

To install this plugin just copy the .php file to your wp-content/plugins folder, or use WordPress Dashboard to install this plugin by searching for it from Plugins -> Add New.
